# Сайт Федерации каратэ г. Новокузнецк

Статический сайт для размещения на GitHub Pages. Все данные подгружаются из CSV-файлов — вы можете редактировать их самостоятельно без изменения кода.

## Структура сайта

- **Главная** — приветствие от президента, последние новости, ближайшие мероприятия
- **Сборная** — карточки спортсменов, сгруппированные по возрастным категориям
- **Тренеры** — карточки тренеров с заслугами и клубами
- **Календарь** — хронология мероприятий с документами для скачивания
- **Фотогалерея** — альбомы мероприятий с фото и ссылками на Яндекс.Диск
- **Новости** — лента новостей в хронологическом порядке
- **Документы** — устав, протоколы, нормативные документы (сгруппированы по категориям)

## Как разместить на GitHub Pages

1. Создайте новый репозиторий на GitHub (можно публичный).
2. Загрузите **все файлы и папки** из этого архива в корень репозитория.
3. Перейдите в раздел **Settings → Pages** (Настройки → Страницы).
4. В разделе "Build and deployment" выберите:
   - Source: **Deploy from a branch**
   - Branch: **main** (или master) / папка **/(root)**
5. Нажмите **Save**. Через 1–2 минуты сайт будет доступен по адресу `https://ваш-ник.github.io/название-репозитория/`

## Как обновлять данные

Все данные хранятся в папке `data/` в формате CSV. Редактируйте их прямо на GitHub (кнопка "Edit this file") или через Excel / Google Таблицы с последующей выгрузкой в CSV (кодировка UTF-8, разделитель — запятая).

### Файл `data/team.csv`

Столбцы: `name,age_category,weight_category,club,coach,results,photo_url`

| Столбец | Описание |
|---------|----------|
| name | ФИО спортсмена |
| age_category | Возрастная категория, например: "Юноши 14-15 лет" |
| weight_category | Весовая категория, например: "70 кг" |
| club | Название клуба |
| coach | ФИО личного тренера |
| results | Последние результаты (через запятую) |
| photo_url | Путь к фото, например: `./media/team/ivanov.jpg` |

**Важно:** спортсмены автоматически группируются по полю `age_category`.

### Файл `data/calendar.csv`

Столбцы: `date,title,location,documents`

| Столбец | Описание |
|---------|----------|
| date | Дата в формате ДД.ММ.ГГГГ или ГГГГ-ММ-ДД |
| title | Название мероприятия |
| location | Город |
| documents | JSON-массив документов. Пример: `[{"name":"Положение","url":"./docs/file.pdf"}]` |

**Важно:** мероприятия сортируются по дате автоматически, независимо от порядка строк в файле.

### Файл `data/coaches.csv`

Столбцы: `name,club,category,achievements,photo_url`

| Столбец | Описание |
|---------|----------|
| name | ФИО тренера |
| club | Клуб |
| category | Категория / дан |
| achievements | Заслуги и достижения |
| photo_url | Путь к фото |

### Файл `data/gallery.csv`

Столбцы: `date,title,photo_urls,yandex_disk_link`

| Столбец | Описание |
|---------|----------|
| date | Дата мероприятия |
| title | Название мероприятия |
| photo_urls | Пути к фото через точку с запятой: `photo1.jpg;photo2.jpg;photo3.jpg` |
| yandex_disk_link | Ссылка на Яндекс.Диск (если есть) |

### Файл `data/news.csv`

Столбцы: `date,title,content,image_url`

| Столбец | Описание |
|---------|----------|
| date | Дата публикации |
| title | Заголовок |
| content | Текст новости |
| image_url | Путь к изображению (можно оставить пустым) |

**Важно:** новости сортируются по дате от новых к старым автоматически.

### Файл `data/documents.csv`

Столбцы: `category,title,file_url,date`

| Столбец | Описание |
|---------|----------|
| category | Категория: "Устав", "Протоколы", "Нормативные документы" и т.д. |
| title | Название документа |
| file_url | Путь к файлу: `./docs/имя_файла.pdf` |
| date | Дата документа |

**Важно:** документы группируются по полю `category`.

## Как добавить фото и документы

1. Загрузите файлы в соответствующие папки:
   - Фото спортсменов → `media/team/`
   - Фото тренеров → `media/coaches/`
   - Фото мероприятий → `media/gallery/`
   - Фото новостей → `media/news/`
   - Фото президента → `media/president/` (замените блок в `index.html` или укажите путь в CSS)
   - PDF/DOC/XLSX документы → `docs/`
2. Укажите пути в соответствующих CSV-файлах.

## Как добавить логотип

Замените блок `<div class="logo-placeholder">` в файле `index.html` (и в других страницах, если нужно) на:

```html
<img src="./media/logo.png" alt="Логотип" style="width:60px;height:60px;object-fit:contain;">
```

Или отредактируйте CSS, чтобы логотип выглядел так, как вам нужно.

## Как изменить приветствие президента

Откройте `index.html`, найдите блок с классом `president-text` и отредактируйте текст. Замените фото, указав путь в блоке `president-photo`:

```html
<img src="./media/president/foto.jpg" alt="Президент" style="width:100%;height:100%;object-fit:cover;border-radius:10px;">
```

## Технические детали

- Сайт полностью статический (HTML + CSS + JS).
- Данные загружаются через `fetch()` из CSV — требуется интернет только для первой загрузки.
- Все сортировки (новости, календарь) происходят в браузере.
- Дизайн адаптивен: работает на смартфонах, планшетах и ПК.
- Для корректной работы с русским языком CSV-файлы должны быть в кодировке **UTF-8**.

## Поддержка

Если нужно добавить новый раздел или изменить дизайн — отредактируйте HTML/CSS файлы или обратитесь за помощью.
